## 🚀 Quick Start

```bash
# Install FastAPI and server
pip install fastapi uvicorn

# Clone this repository and navigate to any section
cd 01-introduction

# Start your first API
uvicorn main:app --reload

# Visit http://localhost:8000/docs - You just built an API! 🎉
```
